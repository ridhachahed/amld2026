Overview

Turn your database into an agent-ready foundation.

This intensive hands-on workshop takes you on a complete journey from cloud fundamentals to building production-ready AI agents powered by MySQL HeatWave GenAI.

You’ll start with foundational concepts of cloud computing and AI agents, then dive deep into MySQL Studio through interactive labs covering SQL, machine learning, GenAI queries, and vector operations.

You’ll learn to create intelligent agents that seamlessly combine structured database queries with unstructured documents through advanced RAG capabilities.

What you’ll build
A fully functional RAG-powered chatbot.
  • Query database tables using natural language (e.g., “Show me top customers from last quarter”)
  • Answer questions from uploaded documents (PDFs, docs, presentations)
  • Combine structured and unstructured data for comprehensive insights
  • Automatically generate and execute SQL from conversational requests
Key technical highlights
Enterprise-focused, production-minded.
  • Configure multi-model AI access through OCI integration
  • Build RAG pipelines combining database tables with document knowledge
  • Implement natural language to SQL translation for intuitive data access
  • Create context-aware chatbots with citation tracking
  • Deploy production-ready solutions with enterprise security
Models, one parameter away
We’ll explore MySQL HeatWave’s OCI GenAI integration, letting you switch between models (Cohere Command, Llama, GPT‑Oss, Gemini, Grok) with a single parameter change — no complex pipelines, no data movement, no separate vector databases.
Real-world applications
Where this pattern pays off immediately.
  • Customer service automation
  • Business intelligence interfaces
  • Technical documentation assistants
  • Compliance Q&A systems
Who is this for?
Perfect fit for these roles.
Developers, data engineers, and architects seeking practical AI integration without infrastructure complexity.
